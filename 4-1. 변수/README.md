# 4. 파이썬 기본문법
## 변수
변화할 수 있는 값
## 변수에 리터럴을 저장하는 것에 대한 도식화
메모리(x variable) <- literal
## 변수 명명 규칙
snake_case를 활용하여 명명하도록 권장
## Snake case
변수명에 공백을 포함할 수 없으므로 언더바를 사용하는 방식
## Pascal case
클래스의 명칭에서 쓰임
## Camel case
자바 필드명 권장사항
## 변수 삭제
del x
## q변수 다뤄보기
```python
a = 10
a = a + 20
print(a)
del a
print(a)
```
## 변수에 빈 값 할당하기
보통 다른언어의 null과 같음
```python
a = None
print(a)
```
## 상수(constant)
파이썬에선 예약된 키워드를 통해 상수를 만드는 방법은 없음
보통 대문자로만 SNAKE_CASE를 활용하여 상수처럼 변수를 활용 가능
