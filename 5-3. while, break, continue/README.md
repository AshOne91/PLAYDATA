## while문
조건식으로만 동작
```
while i < 100:
  print("hello")
  i += 1
```
## while문으로 시퀀스 객체의 인덱스 루프 테크닉
```
while idx < len(scores) :
  print(scores[idx])
  idx += 1
```

## break문
for, while문에서 break는 해당 반복문을 완전히 빠져나감

## while문 - 무한 루프의 활용
```
while True:
  ...
  if i == 100:
    break;
```
